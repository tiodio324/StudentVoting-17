# Учетные данные для входа в систему

## Роли пользователей

Система поддерживает следующие роли:

- **user** (Гость) - роль по умолчанию, не требует авторизации
- **moderator** (Модератор) - требуется авторизация с паролем
- **admin** (Администратор) - требуется авторизация с паролем

## Пароли для входа

| Роль | Пароль |
|------|--------|
| Модератор (moderator) | `moderator2026-voting` |
| Администратор (admin) | `admin2026-voting` |
